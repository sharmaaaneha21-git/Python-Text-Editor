# Python Text Editor

A simple Text Editor built using Python and Tkinter.  
This project allows users to create, open, and save text files using a graphical user interface.

## Features

- Create a new file
- Open existing text files
- Save text files
- Menu bar with File options
- Keyboard shortcuts (Ctrl + N, Ctrl + O, Ctrl + S)

## Technologies Used

- Python
- Tkinter (GUI Library)

## How to Run the Project

1. Clone the repository

git clone https://github.com/sharmaaaneha21-git/Python-Text-Editor.git

2. Go to the project folder

cd Python-Text-Editor

3. Run the program

python main.py
